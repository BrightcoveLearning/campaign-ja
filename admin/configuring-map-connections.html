マーケティング自動化プラットフォーム接続の構成

{{ page.description }}

™Brightcove Campaign は、HubSpot、Eloqua、Marketoなどの一般的なマーケティングオートメーションプラットフォーム（MAP）と統合するように構成できます。MAP 統合を設定すると、Brightcove Campaign を使用して公開された動画の動画エンゲージメントデータは、選択したマーケティング自動化プラットフォームに同期されます。

注:管理メニューにアクセスするには、アカウント管理者である必要があります。
注:接続プロセスにより、ブラウザでポップアップウィンドウが開きます。接続を設定する前に、ブラウザのポップアップブロッカーを無効にする必要があります。

マーケティング自動化プラットフォームとの統合を構成するには、次の手順に従います。

  1. Brightcove Campaign にログインします。
  2. ナビゲーションヘッダーで、歯車アイコン ()、[ MAP 接続] の順に指定します。現在の接続のリスト (存在する場合) が表示されます。
  3. [新規コレクションの追加]をクリックします。
  4. 接続の種類を選択します。そのプラットフォームへの接続については、該当するトピックを参照してください。
    • エロクア
    • マルケト（REST APIまたはマンチキン）
    • ハブスポット ( REST API またはクライアントサイド API )
    • セールスフォース
