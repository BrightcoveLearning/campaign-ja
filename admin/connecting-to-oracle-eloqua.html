Oracle Eloquaに接続中

{{ page.description }}

Brightcoveキャンペーンは、Oracle Eloqua と統合するように設定できます。統合を設定すると、Brightcove Campaign を使用して公開された動画の動画エンゲージメントデータは Eloqua に同期されます。Eloqua では、このデータは、セグメンテーション、キャンペーンロジック、レポート、リードスコアリング、コミュニケーションのパーソナライズに使用できます。

必要条件

Brightcoveキャンペーンを Oracle Eloqua で使用するには、次の条件を満たす必要があります。

  • Oracle Eloqua 10 とログインする資格情報を持っている: https://login.eloqua.com
    注:これらの認証情報は Eloqua ログインページに入力され、ブライトコーブに直接提供されることはありません。
  • Eloqua へのログインに使用する認証情報は、上級ユーザー-マーケティンググループに追加され、次の権限を持っている必要があります。
    • APIを消費
    • クラウドアプリケーションのインストール
    • マーケティングキャンペーンの設定

特徴

ブライトコーブ・キャンペーンの主な機能-Oracle Eloqua との統合には次のものがあります。

  • データ・フォーマット -ビデオ表示データは、Oracle Eloqua内のカスタム・データ・オブジェクト（CDO）に配信されます。Eloqua に複製されるデータは次のとおりです。
    • ビデオ名
    • ビデオ ID
    • ブライトコーブのアカウント ID
    • 参照ページのページ URL
    • ブライトコーブプレイヤー ID
    • % 監視しました
  • データ転送 -15 分以上経過したデータの閲覧は、Brightcove Campaign から Eloqua に 1 時間に 4 回送信されます。これは、データの表示がEloquaに表示されるまでに最大30分かかることを意味します。

接続のセットアップ

Eloqua との統合を設定する前に、新しいブラウザウィンドウを開き、Eloqua アカウントから手動でログアウトすることをお勧めします。これにより、インテグレーションの設定時に Eloqua アカウントにログインするように促され、正しいアカウントを使用していることを確認できます。

注:接続プロセスにより、ブラウザでポップアップウィンドウが開きます。接続を設定する前に、ブラウザのポップアップブロッカーを無効にする必要があります。

次の手順に従って、Brightcoveキャンペーンと Oracle Eloqua 間の接続を設定します。

  1. Brightcove Campaign にログインします。
  2. ナビゲーションヘッダーで、歯車アイコン ()、[ MAP 接続] の順に指定します。現在の接続のリスト (存在する場合) が表示されます。
  3. [ 新しい接続を追加] をクリックします。
  4. 選択するエロクアドロップダウンリストから。
  5. 「Eloqua に接続」をクリックします。
  6. Eloqua にサインインするように求められます。Eloqua の会社名、ユーザー名、パスワードを入力します。[ サインイン] をクリックします。
  7. ログインに成功すると、 Eloqua 用 Brightcove 動画コネクトをインストールするよう求められます。[ 同意してインストール ] をクリックします。
  8. アカウントの Eloqua を有効にするよう求められます。[ 有効 ] をクリックします。
  9. 統合の使用条件を確認し、[ 同意する ] をクリックします。
  10. Eloqua の Brightcove ビデオコネクトが Eloqua アカウントにアクセスすることを許可するように求められます。[ サインイン] をクリックします。
  11. 会社名とユーザー名を確認し、[ Accept ] をクリックします。
  12. 接続の詳細表示を確認します。

Eloqua への接続が設定され、使用できる状態になりました。デフォルトでは、接続ステータスになります有効。接続ステータスによって、動画エンゲージメントデータがEloquaに送信されるかどうかが決まります。

  • 有効 -データが収集され、Eloquaに送信されます
  • 無効 -データは収集されません（統合は無効です）

[ 保存 ] をクリックして変更を保存します。統合を削除するには、[ 削除] をクリックします。
