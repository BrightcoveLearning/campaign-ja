メディア共有設定の管理

{{ page.description }}

メディア共有を使用すると、Video Cloud パブリッシャーは動画を一度アップロードしてから、他のアカウントで動画を共有できます。アカウントは、自分の、法人アカウント、または他のパブリッシャーのアカウントにすることができます。共有動画は、他のアカウントで個別に使用できます。メディア共有設定は、アカウント間の共有関係を管理するために使用されます。

このトピックでは、次の用語を使用します。

  • マスター -元の動画を作成（アップロード）したアカウント。マスターはコンテンツを所有し、アフィリエイトへのコンテンツの設定、管理、提供を担当します。マスターアカウントは常に Brightcove Video Cloud アカウントになります。
  • アフィリエイト -動画を受信しているアカウント。アフィリエイトは、マスターから共有されたコンテンツを受け入れることができます。™Brightcove Campaign アカウントは常にアフィリエイトアカウントです。

アフィリエイトの追加

Brightcove Video Cloud アカウントから Brightcove Campaign アカウントに動画を共有するには、Campaign アカウントを Video Cloud アカウントのアフィリエイトとして追加する必要があります。アカウントを追加する必要があります。Video Cloud アカウントにアフィリエイトアカウントを追加する方法については、 Video Cloud のドキュメントを参照してください。

アフィリエイトアカウントが追加されると、アフィリエイトアカウントはメディア共有通知を受け取り、関係を承諾または拒否することができます。

メディア共有関係の承認/拒否

Brightcove Video Cloud アカウントが Brightcove Campaign アカウントをアフィリエイトとして追加した場合（アカウントに動画を共有したい場合）、メディア共有管理ページにアクセスするとメッセージが表示されます（歯車アイコン（) とメディア共有 )。

メディア共有関係を拒否するには、[拒否] をクリックします。[ 承認 ] をクリックして、メディア共有関係を承認します。関係が承認されると、ページの「マスターの管理」セクションに表示されます。

共有関係を削除するには、削除アイコン（)。
