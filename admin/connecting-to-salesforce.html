    Salesforce に接続中

    このトピックでは、Salesforce で使用するために Brightcove Campaign™ を設定する方法について説明します。

    ブライトコーブキャンペーンは、Salesforce と統合するように設定できます。統合が設定されると、Brightcove Campaign を使用して公開された動画の動画エンゲージメントデータは Salesforce に同期されます。Salesforce では、このデータは、セグメンテーション、キャンペーンロジック、レポート、リードスコアリング、およびコミュニケーションのパーソナライズに使用できます。

    要件

    Salesforce で Brightcove キャンペーンモジュールを使用するには、次の操作を行う必要があります。

    • Salesforce アカウントを持っている、Salesforce サンドボックスアカウントはサポートされていません
    • API が有効な権限を持つ Salesforce.com アカウントを持ち、表示データの送信先として選択した Salesforce オブジェクトの読み取りおよび書き込みを許可するロールに属している
    • Pardot 統合を使用する場合は、「Pardot 統合用のカスタム Salesforce オブジェクトの作成」トピックで説明されている手順を完了する必要があります。

    機能

    ブライトコーブキャンペーン-Salesforce 統合の主な機能は次のとおりです。

    • データ形式 -データの表示は Salesforce オブジェクトにマッピングされます
    • データ転送 -表示データは Brightcove キャンペーンから Salesforce に 1 時間ごとに送信されます
    • リードフォーム -視聴者が動画を視聴したときに自動的に表示される Salesforce フォームを作成します。詳細については、「 Salesforce でのリードフォームの作成」を参照してください。

    接続を設定する

    次の手順に従って、ブライトコーブキャンペーンと Salesforce 間の接続を設定します。

    注：接続プロセスは、ブラウザでポップアップウィンドウを開きます。接続を設定する前に、ブラウザのポップアップブロッカーを無効にする必要があります。
    注：ブライトコーブキャンペーンとの統合を設定するときに使用する専用の Salesforce ユーザアカウントを作成することをお勧めします。
    1. ブライトコーブキャンペーンにログインします。
    2. ナビゲーションヘッダーで、歯車アイコン ( ) をクリックし、[ マップ接続 ] をクリックします。現在の接続（存在する場合）のリストが表示されます。
    3. [新規コレクションの追加]をクリックします。
    4. ドロップダウンリストから Salesforce を選択し、[ Salesforce に接続 ] をクリックします。
    5. 表示されたログイン画面で、Salesforceのユーザー名とパスワードを入力します。[ ログイン ] をクリックします。
    6. ログインが成功すると、Brightcove Video Connect for Salesforce アプリをインストールするようメッセージが表示されます。[許可]をクリックします。
    7. 接続が成功したことを示すメッセージが表示されることを確認します。Brightcove Campaign にリダイレクトされます。
    8. （オプション）接続名を入力します。
    9. 統合タイプを選択します。
      • Pardot -Pardot はビューアを識別するために使用されます。以下の「 Pardot 統合の設定」セクションを参照してください。
      • 手動 -開発スタッフが、視聴者を識別するためのロジックを追加する責任があります。
    10. [ マッピングタイプ ] を選択し、データマッピングを構成します。
    11. [保存]をクリックします。
    12. ページ上部の [ マップ接続に戻る ] リンクをクリックして、[マップ接続] ページに戻ります。
    注：統合タイプを構成せずに接続を作成すると、[マップ接続] ページで、接続のステータスは [ 接続不完全 ] になります。

    Pardot 統合の設定

    ブライトコーブキャンペーンから Salesforce への接続が完了したら、統合タイプを設定する必要があります。このセクションでは、 Pardot 統合タイプを使用する手順について説明します。

    注：Pardot 統合を設定する前に、「 Pardot 統合用のカスタム Salesforce オブジェクトの作成」トピックで説明されている手順を完了する必要があります。
    注：ブライトコーブキャンペーンとの統合を設定するときに使用する専用の Pardot ユーザーアカウントを作成することをお勧めします。
    1. 統合タイプとして Pardot を選択します。
    2. Pardotへのログインに使用するEメールとパスワードを入力します。

    Pardot API キーを取得する

    [API Key] フィールドの値を取得するには、次の手順に従います。

    1. Pardot にログインします。
    2. 歯車アイコン ( ) をクリックし、[ 設定] をクリックします。
    3. ページの上部にある [ マイプロフィール ] リンクをクリックします。
    4. [ API User Key ] フィールドの値をコピーし、[ User Key ] フィールドに貼り付けます。

    トラッキング ID を取得する

    トラッキング PiLd フィールドとトラッキング PiCd フィールドの値を取得するには、次の手順を実行します。

    1. 左側のナビゲーションで [ マーケティング ] アイコンをクリックし、[ キャンペーン ] をクリックします。
    2. キャンペーン名をクリックします（キャンペーンがない場合は、キャンペーンを追加します）。
      注：どのキャンペーンを選択しても構いません。ブライトコーブキャンペーンでは、閲覧者を識別するために Cookie を取得するために両方の値が必要です。
    3. [ トラッキングコードを表示 ] をクリックします。PiaID と picID の値が表示されます。
    4. 値をコピーし、それぞれのフィールドに貼り付けます。
      注：ブライトコーブキャンペーンでは、PiaID フィールドと PicID フィールドは検証されませんが、間違った値が入力されると、ブライトコーブキャンペーンは閲覧者を識別できません。
    5. [ Pardot に接続 ] をクリックします。

    通常、Pardot 統合タイプを使用する場合は、[ 高度なマッピングタイプ ] を選択し、作成した Salesforce カスタムオブジェクトにビュー項目をマップします。

    データマッピングを設定する

    動画データがSalesforceオブジェクトのフィールドにどのようにマッピングされるかが、データマッピングで決まります。デフォルトでは、統合は簡易マッピングを使用して設定されます。この場合、Salesforceの Task オブジェクトに動画データがマッピングされます。

    以下の手順に従って、データマッピングをカスタマイズします。

    1. [マッピングタイプ]に[詳細]を選択します。
    2. [ Salesforce オブジェクト ] ドロップダウンリストで、ビデオデータをマップする Salesforce オブジェクトを選択します。通常、Pardot 統合タイプを使用する場合は、作成した Salesforce カスタムオブジェクトにビュー項目をマップします。

    3. 適切なビュー項目を Salesforce 項目にマップします。次の例は、カスタムオブジェクトにマップされるビデオフィールドを示しています。
    4. [保存] をクリックして、変更を保存します。

    Salesforce との統合を削除するには、[ 削除 ] をクリックします。

    トラッキングIDを渡す

    ブライトコーブキャンペーンでは、トラッキング ID 値を Salesforce に渡す機能をサポートしています。トラッキング ID は、たとえば、キャンペーン ID の追跡に使用されます。トラッキング ID は、動画を表示するページ URL に追加するか、埋め込みコードに追加できます。

    • 例:URLに追加
      http://www.myurl.com/video.html？ bctrackingid=12345
    • 例:プレーヤーの埋め込みコードに追加しました
      <video data-video-id= "6033402539001" 
         data-account= "1486906377" 
         データプレーヤー=「デフォルト」 
         データ埋め込み=「デフォルト」 
         data-application-id 
         class= "video-js」
         data-bc-trackingid= "12345"  
         > <コントロール/ビデオ >
  <スクリプト src=」//players.brightcove.net/1486906377/default_default/index.min.js」> < /script >

    その後、Brightcove Campaign は、同期される別の項目として Salesforce に値を渡します。

    視聴者を識別する

    Salesforce 接続が [ 手動 ] 統合タイプで構成されている場合、閲覧者を識別するのはユーザー次第です。視聴者をトラッキングするには、動画を表示しているページでユーザーIDをURLパラメーターとして渡す必要があります。2つのURLパラメーターがサポートされています。

    • bcemail - 視聴者の電子メールアドレスを渡します
    • bcsfid - 視聴者のSalesforce IDを渡します

    このシナリオでは、トラッキング用に一意に生成されたURLを使って、動画ランディングページが生成されます。その後、Brightcove Campaign は URL パラメータを使用してユーザーのサーバー側を検索し、閲覧データを同期します。

    Salesforceで動画データを表示する

    動画が閲覧されると、動画視聴データは Salesforce に同期されます。簡易データマッピングオプションが選択されている場合、動画ビューデータは、Salesforce内部で Task 項目として現れます。詳細データマッピングオプションが使用されている場合、視聴データは、マッピングが設定されたときに選択されたSalesforceオブジェクトの一部として現れます。次の例では、高度なデータマッピングが使用され、ビデオ項目がカスタム BC ビデオビュー Salesforce オブジェクトにマッピングされています。

    動画名をクリックすると、詳細情報が表示されます。
