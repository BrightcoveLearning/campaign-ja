動画を公開する

{{ page.description }}

Brightcove Campaign ™の公開機能を使用すると、マーケティングオートメーションプラットフォーム (MAP) を使用して動画を配信するために必要な埋め込みコードと画像を簡単に生成できます。通常、埋め込みコードは、マップで作成された HTML ランディングページに貼り付けられ、ランディングページへのリンクが電子メールで送信されます。

Web ページへの公開

動画の埋め込みコードをコピーするには、次の手順に従います。

  1. [ ビデオ] ページを開きます。
  2. ビデオの横にあるチェックボックスをクリックして、ビデオを選択します。
  3. [ パブリッシュ] をクリックします。
  4. （オプション） MAP 接続を選択します。ドロップダウンリストには、作成されたすべての MAP 接続のリストが含まれます。MAP 接続を選択すると、選択したプラットフォームにビデオ視聴アクティビティが送信されます。このオプションを使用できるかどうかは、使用されているBrightcove Campaign パッケージに基づいています。
  5. （オプション）リードフォームを選択します。ドロップダウンリストには、作成されたすべてのリードフォームのリストが含まれます。MAP 接続を選択すると、ビデオの再生中にリードフォームが表示され、視聴者の情報をキャプチャできます。このオプションを使用できるかどうかは、使用されているBrightcove Campaign パッケージに基づいています。
  6. 埋め込みコードをコピーし、ランディングページに貼り付けます。これは、Webページにビデオを公開するために必要なコードです。
  7. ビデオ SEO コードをコピーし、<head></head>ランディングページのタグとタグの間にコードを貼り付けます。ビデオ SEO コードは、検索エンジンへのランディングページの可視性を向上させるのに役立ちます。ページが公開されると、Google は URL 検査ツールを提供します。このツールを使用すると、サイト上の URL に関する Google インデックスに関する詳細情報を表示できます。
  8. Facebook タグをコピーし、<head></head>ランディングページのタグとタグの間にコードを貼り付けます。Facebookのタグは、ページがFacebookに共有されたときに表示されるものを制御するのに役立つFacebookオープングラフタグです。

ポスター画像のコピー

ほとんどのプラットフォームでは、’電子メールに直接動画を埋め込むことは不可能です。これはブライトコーブの制限ではなく、セキュリティ上の理由から、ほぼすべての最新のメールシステムがビデオプレーヤーの埋め込みコードを取り除きます。ただし、ティーザーテキストとポスター画像を含むメールを作成することで、コンテンツの認知度とトラフィックを促進できます。ポスター画像は動画プレーヤーのように見え、動画のランディングページにリダイレクトするクリック可能なリンクになります。

ビデオのポスターイメージをコピーするには、次の手順に従います。

  1. クリックEメールページ上部のタブ。
  2. [ コピー ] をクリックして、ポスターイメージをコピーします。
  3. メールを送信するツールに画像を貼り付けます。
  4. 公開されたビデオを使用して、画像からランディングページへのハイパーリンクを作成します。
