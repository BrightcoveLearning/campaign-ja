電子メールによるビデオビューの促進

{{ page.description }}

™Brightcove Campaign の公開機能を使用すると、ランディングページと電子メールを使用してビデオを配信するために必要な埋め込みコードと画像を簡単に生成できます。通常、動画の埋め込みコードはランディングページに貼り付けられ、ランディングページへのリンクが電子メールで送信されます。このトピックで説明するプロセスは、マーケティング自動化プラットフォーム (MAP) が使用されていない場合に最適です。MAP を使用している場合、ブライトコーブでは通常の公開プロセスを使用することをお勧めします。

ランディングページの作成

ランディングページは、製品やサービスを宣伝するためのマーケティングキャンペーンの一部として使用できるスタンドアロンの Web ページです。ランディングページに動画を追加することは、視聴者のエンゲージメントを高め、視聴者を購入者に変えるのに最適な方法です。ランディングページを作成する機能を提供するベンダーは多数あります。ランディングページは、マーケティングオートメーションプラットフォームソフトウェアを使用して作成することも、開発者リソースがある場合は、カスタム HTML を使用して手動で作成することもできます。ランディングページを作成したら、Brightcove Campaign を使用してランディングページに動画を埋め込むことができます。

埋め込みコードの生成

動画がランディングページで再生されるようにするには、埋め込みコードが必要です。ビデオの埋め込みコードをコピーするには、次の手順に従います。

  1. [ ビデオ] ページを開きます。
  2. 使用するビデオの横にあるチェックボックスをクリックして、ビデオを選択します。
  3. [ パブリッシュ] をクリックします。
  4. をコピーします埋め込みコード動画を表示するランディングページに貼り付けます。
  5. ランディングページへの変更を保存します。
  6. ランディングページを公開し、ランディングページを表示できる URL があることを確認します。

ポスター画像のコピー

ほとんどのプラットフォームでは、’電子メールに直接動画を埋め込むことは不可能です。これはブライトコーブの制限ではなく、セキュリティ上の理由から、ほぼすべての最新のメールシステムがビデオプレーヤーの埋め込みコードを取り除きます。しかし、ティーザーテキストとポスター画像を含むメールを作成することで、コンテンツの認知度とトラフィックを促進することができます。ポスター画像は動画プレーヤーのように見え、ランディングページにリダイレクトするクリック可能なリンクになります。

ビデオのポスター画像をコピーするには、次の手順に従います。

  1. クリックEメールページ上部のタブ。
  2. [ コピー ] をクリックして、ポスターイメージをコピーします。
  3. メールを送信するツールに画像を貼り付けます。
  4. 画像からランディングページへのハイパーリンクを作成します。これを行うプロセスは、使用されている電子メールプログラムによって異なります。

メールが送信されると、読者は電子メールの一部として動画が表示され、画像をクリックするとランディングページに誘導され、そこで動画を見ることができます。
