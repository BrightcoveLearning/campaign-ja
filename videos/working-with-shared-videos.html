共有ビデオの操作

{{ page.description }}

メディア共有では、パブリッシャーが動画を一度アップロードし、その動画を他のアカウントと共有できます。アカウントには、自分のアカウント、法人アカウント、または他のパブリッシャーのアカウントを使用できます。共有動画は、他のアカウントで個別に使用できます。メディア共有は通常、パブリッシャーが動画コンテンツのマスターアカウントを保持し、組織の他の部門や子会社に動画を共有したい場合に使用されます。

メディア共有の用語

このトピックでは、次の用語を使用します。

  • マスター -元のビデオを作成したアカウント。マスターはコンテンツを所有し、アフィリエイトへのコンテンツの設定、管理、提供を担当します。マスターアカウントは、常に Brightcove 動画クラウドアカウントになります。
  • アフィリエイト -動画を受信しているアカウント。アフィリエイトは、マスターから共有されたコンテンツを受け入れることができます。™Brightcove Campaign アカウントは常にアフィリエイトアカウントです。

メディア共有関係の管理

動画を Brightcove Video Cloud アカウントから Brightcove Campaign アカウントで共有するには、Video Cloud アカウントでメディア共有設定を設定する必要があります。これを行う方法については、 Video Cloud のドキュメントを参照してください。

Video Cloud アカウントが Campaign アカウントをアフィリエイトとして追加した後、アフィリエイトアカウントはリレーションシップを受け入れる必要があります。マスターアカウントとアフィリエイトアカウントは、いつでもリレーションシップを終了できます。メディア共有関係を終了しても、アフィリエイトアカウントから共有動画は削除されません。

メディア共有設定の詳細については、「メディア共有設定の管理」を参照してください。

動画を共有する

Brightcoveキャンペーンアカウントで共有される動画はすべて、Brightcove Video Cloud アカウントから発信されます。Video Cloud アカウントから動画を共有する方法については、 Video Cloud のドキュメントを参照してください。

注:Video Cloud から Brightcove Campaign で動画を共有する場合は、動画の [ 広告サポート ] フィールドが [ 無料 ] に設定されていることを確認してください。そうすると共有エラーが発生します。

[動画] ページの共有動画

共有動画は、動画のサムネイルの左上隅に共有アイコンが表示され、その動画がアカウントで共有されたことを示します。

動画のタイトルをクリックし、動画情報のプロパティを確認して、動画を共有したアカウントを確認します。

共有ビデオのプロパティ

動画をアフィリエイトアカウントと共有すると、次の動画プロパティが共有動画とともにコピーされます。

  • 名前
  • 短い説明
  • タグ
  • サムネイルとポスター画像-デフォルトでは、マスタービデオのポスターとサムネイル画像はアフィリエイトアカウントで使用されます。マスターアカウントのポスターおよび/またはサムネイルに変更が加えられた場合、アフィリエイトアカウントが新しいポスターまたはサムネイル画像を作成していない限り、それらの変更はアフィリエイトアカウントに反映されます。アフィリエイトアカウントがポスターまたはサムネイル画像を変更すると、マスターアカウントで行われたポスター/サムネイルの変更は、アフィリエイトアカウントに反映されません。
  • テキストトラック-アフィリエイトは、共有ビデオのテキストトラックのプロパティを削除または編集することはできません。

共有ビデオには、マスタービデオが共有された時点で持っていたこれらのビデオメタデータフィールドのすべての値が含まれています。ただし、その時点から、マスターコピーとアフィリエイトコピーは独立したエンティティになります。

  • メディア共有を使用する場合、ビデオファイルはコピーされず、ビデオメタデータのみがコピーされます。共有動画は、アカウント内の元の動画ファイルを指しています。
  • 動画がマスターアカウントから削除されると、その動画はすべてのアフィリエイトアカウントから削除されます。
  • アフィリエイトが動画に加えた変更は、オリジナルには影響しません。アフィリエイトが共有動画を削除した場合、その動画はアフィリエイトのアカウントから削除されますが、マスターアカウントに残ります。
  • アフィリエイトはテキストトラックを削除したり更新したりすることはできません。
  • アフィリエイトが共有して動画を受信すると、その動画はアクティブ状態になります。これは、ビデオがマスターアカウントでアクティブでなかった場合でも当てはまります。
  • 各アカウントは、そのアカウントからの共有ビデオの再生（ビデオ再生と帯域幅の使用量）に対して個別に請求されます。
  • 共有動画に関するレポートは各アカウントに固有のものであり、各アカウントの動画の各コピーに関する情報は集計されません。

動画を再共有できます（以前に共有した動画を同じアカウントと共有する）。そうすることで、アフィリエイトアカウントによって行われた動画メタデータの変更が、マスター（共有）アカウントの値に置き換えられます。動画を再共有するときは注意してください。
