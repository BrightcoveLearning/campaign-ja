Brightcove Campaign™ ダッシュボードの概要

{{ page.description }}

Brightcove キャンペーンダッシュボードページには、動画がどのように掲載されているかの概要が表示されます。表示されるダッシュボードページは、アカウントのステータスおよびアカウントの設定方法によって異なります。以下のセクションは、ダッシュボードレイアウトの例を示しています。アカウントに表示されるものと一致するダッシュボードを見つけ、そのダッシュボードの使用方法の詳細を参照するリンクをクリックしてください。

高度なダッシュボード

Advanced Dashboard には、アカウント内の最高パフォーマンス動画と最低パフォーマンスの動画の概要が表示されます。このダッシュボードには、各動画のビデオベンチマークスコア（VBS）が表示されます。VBS は、動画のパフォーマンスと、同じタイプの他の類似動画のパフォーマンスを比較します。詳細ダッシュボードは、動画の [ 動画コンテンツタイプ] メタデータフィールドを有効にしているアカウントにのみ表示されます。この設定は、アカウント設定の一部として有効にすることができます。

詳細ダッシュボードの詳細は、「詳細ダッシュボードの確認」を参照してください。

基本ダッシュボード

Basic Dashboard には、アカウント内で最もパフォーマンスの高い動画の概要が表示されます。グラフには、動画のインプレッション数、動画開始数、平均視聴率、平均視聴時間が表示されます。このダッシュボードでは、動画ごとに動画ベンチマークスコア（VBS）が提供されないため、動画のパフォーマンスを同じ種類の他の類似動画の掲載結果と比較することはできません。VBS と比較分析を有効にするには、動画の [動画コンテンツタイプ] メタデータフィールドを有効にします。この設定は、アカウント設定の一部として有効にすることができます。

基本ダッシュボードの詳細は、「基本ダッシュボードのレビュー」を参照してください。

スタートダッシュボード

スタートダッシュボードは、Brightcove Campaign アプリケーションに初めてログインしたときに表示され、使用を開始する際に役立ちます。

Brightcove キャンペーンの開始方法の詳細については、「Brightcoveキャンペーンを開始する」を参照してください。
